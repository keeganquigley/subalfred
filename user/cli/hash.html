Subalfred Book

The Hash Command

Hash the data with the specify algorithm.

Available algorithms:

  • blake2-128
  • blake2-128-concat (default)
  • blake2-256
  • blake2-512
  • twox64
  • twox64-concat
  • twox128
  • twox256
  • keccak256
  • keccak512
  • sha2-256

Examples

blake2-128-concat (Default)

subalfred hash 0xb4f7f03bebc56ebe96bc52ea5ed3159d45a0ce3a8d7f082983c33ef133274747
